# PROJETO: NEX FPS COUNTER
Migração Total para Flutter (Dart) com Reformulação Completa de UI, UX, Sistemas e Performance

## 1. Visão Geral do Projeto

Este projeto tem como objetivo realizar a migração completa de um aplicativo Android atualmente desenvolvido em Kotlin para Flutter (Dart), removendo totalmente qualquer dependência de Kotlin ou Java e adotando uma arquitetura moderna, escalável, performática e de fácil manutenção.

O aplicativo será relançado com uma nova identidade visual, uma interface moderna focada no público gamer, novos sistemas internos, maior nível de personalização, animações fluidas e monitoramento avançado de hardware em tempo real, tornando-se uma referência entre aplicativos de contagem de FPS para jogos mobile.
